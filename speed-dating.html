Graph-Based Speed Dating Compatibility App

Interactive dating app using real-world speed dating data and graph algorithms to compute matches.

  • Processed 800+ speed dating records from Kaggle using Pandas.
  • Calculated weighted compatibility using custom NetworkX graph algorithms.
  • Created an interactive Tkinter GUI with match recommendations.
  • Visualized networks with Plotly (dynamic edge weights, color-coded nodes, tooltips).
